# CouncilWatch

CouncilWatch is a public-record governance monitoring layer for Kempsey Shire Council records.

It is separate from any flood, river, bridge or emergency monitoring system.

## Purpose

CouncilWatch answers one core question:

> Council resolved this. Did it happen?

## Scope

CouncilWatch tracks:

- Council resolutions
- Council actions
- Council reports
- Responsible units
- Due dates
- Completion evidence
- Overdue items
- Verification status
- Community-impact matters

## Verification Rule

Proof of Fact remains zero until source records are preserved and checked.

CouncilWatch must not be used as final evidence unless each claim links back to an original council record and the extracted text has been reviewed.
