# Source Targets

## Purpose

Defines the public-record sources the automated fetch pipeline must check.

## Primary Source Group

Authority: Kempsey Shire Council

Target record types:

- Ordinary Council Meeting minutes
- Ordinary Council Meeting agendas
- Ordinary Council Meeting business papers
- Extraordinary Council Meeting records
- Attachments
- Notices of motion
- Public reports
- Media releases
- Public exhibition notices
- Grant-related council records

## Initial URL Capture Fields

Each source target must record:

- Target ID
- Authority
- Source group
- Record type
- URL
- Access method
- Fetch frequency
- Last checked
- Last successful fetch
- Known limitations
- Notes
