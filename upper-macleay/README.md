# Upper Macleay Accountability Repository

## Purpose

This folder is the working accountability framework for Upper Macleay matters.

It preserves public-record evidence, council decisions, timelines, issue maps, source registers, and draft outputs by locality and matter.

## Core Rule

One Matter = One Evidence Chain.

No matter should borrow facts from another matter unless the connection is explicitly recorded and sourced.

## Localities

- Bellbrook
- Willawarrin
- Sherwood
- Turners Flat
- Toorooka
- Regional

## Matter Workflow

Each matter should contain:

- SEARCH_REGISTER.md
- SOURCES_REGISTER.md
- EVIDENCE_REGISTER.md
- DECISION_REGISTER.md
- TIMELINE.md
- ISSUE_MAP.md
- FINDINGS.md
- OUTPUTS.md

## Evidence Standard

No finding is accepted without a traceable source, record, date, document, meeting item, or clearly marked user-supplied statement.
